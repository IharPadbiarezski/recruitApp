# recruitApp
Application to recruit

# prepare and run client

 Go to the client folder 

 ``` cd client\ ```

 Run 
 ``` npm install ```

 Start server
 ``` npm start ```



# prepare and run backend

 Go to the backend folder 

 ``` cd backend\ ```

 Run 
 ``` npm install ```

 Start server
 ``` npm start ```

 Make sure you should have .env file in the folder backend
